# 北京中考英语试卷导入与词汇提取功能说明

## 一、功能概述

本系统是专为北京中考英语教研设计的智能试卷处理平台，核心功能是将纸质或电子版的中考英语试卷自动转化为结构化的词汇数据库。

### 系统能力

| 能力 | 说明 |
|------|------|
| 智能解析 | 自动识别并提取试卷中的阅读理解文章（C篇、D篇）及对应题目 |
| 主题分类 | AI自动分析文章内容，打上主题标签（科技、环保、文化等） |
| 词汇提取 | 从每篇文章中智能提取8-15个核心词汇 |
| 词频统计 | 跨多份试卷统计词汇出现频率，生成高频词库 |
| 例句关联 | 每个词汇都关联到原文中的真实例句 |

### 适用场景

- **教研备课**：快速掌握考试高频词汇，精准定位教学重点
- **词汇教学**：使用真题例句进行词汇讲解，提升教学针对性
- **学生复习**：按词频和主题学习词汇，提高复习效率
- **试卷分析**：追溯词汇来源，分析各区试卷词汇特点

## 二、导入操作流程

## 三、词汇提取详解

### 3.1 提取原则

系统使用AI智能提取，遵循以下原则：

| 原则 | 说明 | 示例 |
|------|------|------|
| 主题相关 | 优先选择与文章主题紧密相关的词汇 | 科技文章优先提取 "innovation" |
| 学习价值 | 选择对中学生有实际学习意义的词汇 | 避免过于简单或过于生僻 |
| 难度适中 | 适合中考水平的词汇 | 避免小学词汇和大学词汇 |
| 用法典型 | 选择在文章中有典型用法的词汇 | 优先选择有完整句子的词汇 |

### 3.2 过滤规则

以下类型的词汇不会被提取：

- **基础词汇**：the, is, are, was, were 等
- **专有名词**：人名、地名、组织名等
- **数字**：数字、年份、日期等
- **重复词汇**：同一词根的不同形式只保留一个

### 3.3 词汇数据结构

每个提取的词汇包含完整的信息：

| 字段 | 说明 | 示例 |
|------|------|------|
| word | 原文出现的具体形式 | discovered |
| lemma | 词根/词元（基础形式） | discover |
| frequency | 在所有试卷中出现的次数 | 12 |
| definition | 中文释义 | v. 发现，发觉 |
| occurrences | 出现位置列表 | 包含句子、位置等信息 |

**出现位置（occurrence）包含：**
- 包含该词的完整句子
- 字符在文章中的起始位置
- 字符在文章中的结束位置
- 该词在句中的位置

## 七、常见问题

### Q1：导入失败怎么办？

**可能原因及解决方案：**

| 原因 | 解决方案 |
|------|----------|
| 文件格式不正确 | 确保是.docx格式 |
| 文件损坏 | 重新保存或修复文件 |
| 文件名包含特殊字符 | 重命名文件，使用中文或英文字母 |
| 网络中断 | 检查网络后重试 |

### Q2：提取的词汇不准确怎么办？

词汇提取由AI完成，可能存在误差。如发现问题：
1. 核对原试卷内容是否清晰
2. 检查文件格式是否正确
3. 联系技术支持反馈问题

### Q3：如何更新已有试卷的词汇？

1. 进入试卷导入页面
2. 勾选"强制重新导入"
3. 重新上传该试卷
4. 系统将覆盖原有数据

### Q4：词汇的词频是实时的吗？

是的，词频统计基于当前数据库中所有试卷的数据。每次导入新试卷后，相关词汇的词频会自动更新。
