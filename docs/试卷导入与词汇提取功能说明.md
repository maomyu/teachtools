# 北京中考英语试卷导入与内容提取功能说明

## 一、功能概述

本系统是专为北京中考英语教研设计的智能试卷处理平台，核心功能是将纸质或电子版的中考英语试卷自动转化为结构化的内容数据库，包括**阅读理解**和**完形填空**两大模块。

### 系统能力

| 能力 | 说明 |
|------|------|
| 智能解析 | 自动识别并提取试卷中的阅读理解文章（C篇、D篇）及对应题目 |
| 完形提取 | 自动识别并提取完形填空文章，解析空格、选项和答案 |
| 主题分类 | AI自动分析文章内容，打上主题标签（科技、环保、文化等） |
| 词汇提取 | 从阅读文章中智能提取8-15个核心词汇 |
| 考点分析 | AI分析完形填空的四类考点（词汇/固定搭配/词义辨析/熟词僻义） |
| 词频统计 | 跨多份试卷统计词汇出现频率，生成高频词库 |
| 考点汇总 | 按考点类型聚合查看所有真题实例 |
| 例句关联 | 每个词汇都关联到原文中的真实例句 |

### 适用场景

- **教研备课**：快速掌握考试高频词汇，精准定位教学重点
- **词汇教学**：使用真题例句进行词汇讲解，提升教学针对性
- **学生复习**：按词频和主题学习词汇，提高复习效率
- **试卷分析**：追溯词汇来源，分析各区试卷词汇特点

## 二、导入操作流程

### 2.1 准备工作

**文件要求：**
- 文件格式：`.docx`（Word 2007及以上版本）
- 文件命名：建议包含年份、地区、年级信息
- 命名示例：`2023年海淀区初三一模英语.docx`

**内容要求：**
- 试卷需包含完整的阅读理解部分
- C篇、D篇阅读理解是主要提取目标
- 题目与文章需对应清晰

### 2.2 导入步骤

#### 步骤一：进入导入页面

在系统导航栏点击"试卷导入"进入导入界面。

#### 步骤二：上传试卷文件

**上传方式：**
- **拖拽上传**：将文件直接拖入上传区域
- **点击上传**：点击上传区域，在文件选择器中选择文件
- **批量上传**：可同时选择多个文件进行批量导入

**可选设置：**
- **强制重新导入**：勾选后，如果数据库中已存在同名试卷，将覆盖原有数据

#### 步骤三：查看处理进度

上传后，系统自动开始处理，您将看到**12个步骤**的实时进度：

| 步序 | 步骤名称 | 处理内容 | 说明 |
|------|----------|----------|------|
| 1 | 上传文件 | 将试卷文件上传至服务器 | 建立文件传输通道 |
| 2 | 解析文件名 | 从文件名提取元数据 | 识别年份、地区、年级等信息 |
| 3 | 上传AI服务 | 将文件发送至AI解析引擎 | 准备进行智能解析 |
| 4 | AI解析文档 | AI识别文章和题目结构 | 核心解析步骤，耗时最长 |
| 5 | 保存阅读文章 | 将阅读C/D篇内容存入数据库 | 结构化存储 |
| 6 | 保存阅读题目 | 将阅读题目内容存入数据库 | 保存答案和选项 |
| 7 | AI阅读话题分类 | AI分析阅读文章主题 | 自动打标签 |
| 8 | 词汇提取 | AI从阅读文章中提取核心词汇 | 生成词库数据 |
| 9 | 保存完形文章 | 将完形填空内容存入数据库 | 保存带空格原文和完整文章 |
| 10 | 保存完形空格 | 将空格、选项、答案、解析存入数据库 | 每个空格独立记录 |
| 11 | AI完形话题分类 | AI分析完形文章主题 | 独立的话题池 |
| 12 | 考点分析 | AI分析每个空格的考点类型 | 四类考点智能识别 |
| 13 | 完形词汇提取 | AI从完形文章中提取核心词汇 | 填入统一词库 |

- 每个步骤有三种状态：等待（灰色）、处理中（蓝色）、完成（绿色）、失败（红色）
- 多个文件并行处理，各自独立显示进度

|----------|--------------|

| 大型试卷（多篇文章） | 90-120秒 |

**时间分布参考**：
- 阅读模块处理：约30-40秒
- 完形模块处理：约30-50秒（含12个空格的考点分析）

## 三、词汇提取详解

### 3.1 提取原则

系统使用AI智能提取，遵循以下原则：

| 原则 | 说明 | 示例 |
|------|------|------|
| 主题相关 | 优先选择与文章主题紧密相关的词汇 | 科技文章优先提取 "innovation" |
| 学习价值 | 选择对中学生有实际学习意义的词汇 | 避免过于简单或过于生僻 |
| 难度适中 | 适合中考水平的词汇 | 避免小学词汇和大学词汇 |
| 用法典型 | 选择在文章中有典型用法的词汇 | 优先选择有完整句子的词汇 |

### 3.2 过滤规则

以下类型的词汇不会被提取：

- **基础词汇**：the, is, are, was, were 等
- **专有名词**：人名、地名、组织名等
- **数字**：数字、年份、日期等
- **重复词汇**：同一词根的不同形式只保留一个

### 3.3 词汇数据结构

每个提取的词汇包含完整的信息：

| 字段 | 说明 | 示例 |
|------|------|------|
| word | 原文出现的具体形式 | discovered |
| lemma | 词根/词元（基础形式） | discover |
| frequency | 在所有试卷中出现的次数 | 12 |
| definition | 中文释义 | v. 发现，发觉 |
| occurrences | 出现位置列表 | 包含句子、位置等信息 |

**出现位置（occurrence）包含：**
- 包含该词的完整句子
- 字符在文章中的起始位置
- 字符在文章中的结束位置
- 该词在句中的位置

## 四、词库管理功能

### 4.1 词库入口

导入完成后，点击导航栏"高频词库"进入词库页面。

> **统一词库设计**：阅读和完形的词汇共用同一入口，支持按来源筛选。

### 4.2 筛选功能

系统提供 **8个维度** 筛选，帮助您精准定位需要的词汇：

| 筛选维度 | 说明 | 示例 |
|----------|------|------|
| **来源** | 按词汇来源筛选 | 全部 / 仅阅读 / 仅完形 |
| 年级 | 按初二、初三筛选 | 初三 |
| 主题 | 按文章主题筛选（支持搜索） | 科技与创新 |
| 年份 | 按试卷年份筛选 | 2023年 |
| 区县 | 按行政区域筛选（支持搜索） | 海淀区 |
| 考试类型 | 按考试类型筛选 | 一模、二模、期中、期末 |
| 学期 | 按学期筛选 | 上学期、下学期 |
| 最低词频 | 设置最低出现次数 | ≥5次 |

**来源筛选说明**：
- **全部**：显示所有词汇，聚合阅读和完形的出现次数
- **仅阅读**：只显示在阅读C/D篇中出现过的词汇
- **仅完形**：只显示在完形填空文章中出现过的词汇

**筛选交互：**
- 所有筛选条件可组合使用
- 当前筛选条件以标签形式显示在统计区域
- 点击标签 × 可快速取消对应筛选
- 点击"重置筛选"清空所有条件

**实时搜索：**
- 搜索框支持精确匹配单词
- 输入后自动筛选表格数据
- 清空搜索框恢复全部显示

### 4.3 词频标识系统

词汇按出现频率用不同颜色标识：

| 颜色 | 词频范围 | 重要程度 | 学习建议 |
|------|----------|----------|----------|
| 红色 | ≥20次 | 核心高频词 | 必须熟练掌握 |
| 橙色 | 10-19次 | 中频词 | 重点学习 |
| 蓝色 | <10次 | 普通词汇 | 了解即可 |

### 4.4 词汇详情查看（核心功能）

**统一词库展示**：


#### 第一步：点击词汇行 → 例句面板

点击表格中任意词汇，右侧滑出**例句面板**（蓝色边框）：

| 信息项 | 说明 |
|--------|------|
| 单词 | 英文原词（大号加粗显示） |
| 释义 | 中文解释 |
| 全局词频 | 在所有试卷中出现的总次数 |
| 例句列表 | 所有关联的阅读文章例句 |

**例句卡片内容：**
- 出处标签（如：2023年海淀区初三一模）
- 例句内容
- 提示文字："点击查看原文"

**分页加载：**
- 首次加载显示前 10 条
- 显示已加载数量（如：已加载 10 / 25 条）
- 点击"加载更多"按钮继续加载

#### 第二步：点击例句卡片 → 文章抽屉

点击任意例句卡片，最右侧滑出**文章抽屉**（绿色边框），展示完整文章信息：

**文章信息区：**
- 文章类型（C篇/D篇）+ 主题
- 出处详情：年份、区县、年级、词数
- "返回例句列表"按钮

**原文内容区：**
- 完整阅读原文
- **目标词汇高亮显示**（黄色背景加粗 = 当前聚焦，浅黄色 = 其他出现位置）
- 导航控件：
  - 显示当前聚焦位置（如：第 1/3 处）
  - "下一处"按钮：跳转到该词下一个出现位置
  - "清除"按钮：取消高亮

**高频词汇标签页：**
- 显示该篇文章的所有核心词汇
- 点击任意词汇可切换高亮
- 显示词汇在本文中的出现次数

**题目标签页：**
- 显示该篇文章对应的所有题目
- 题目内容 + 选项（A/B/C/D）
- "显示答案"按钮：展开正确答案和解析

#### 应用场景示例

假设单词 "environment" 词频为 15，完整操作流程：

1. **点击词汇行** → 右侧出现例句面板，显示 15 个例句
2. **点击第 1 个例句** → 最右侧出现文章抽屉
3. **查看原文** → "environment" 以黄色高亮，共 3 处
4. **点击"下一处"** → 滚动定位到第 2 处出现位置
5. **切换词汇** → 在"高频词汇"列表点击其他词汇，原文高亮切换
6. **查看题目** → 切换到"题目"标签，查看该篇阅读的题目和答案
7. **返回例句** → 点击"返回例句列表"，选择其他出处继续查看

### 4.5 搜索功能

**精确匹配搜索：**
- 在搜索框输入单词

**搜索特点：**
- 精确匹配（不支持模糊搜索）
- 实时筛选表格数据
- 清空搜索框恢复全部显示

**搜索结果操作：**
- 搜索结果直接显示在词汇表格中
- 点击词汇行可查看详情
- 与其他筛选条件可组合使用

**表格列说明：**

| 列名 | 内容 | 说明 |

| 年份 | 出现的年份 | 多个年份用逗号分隔 |
| 区县 | 出现的区县 | 最多显示3个，超出显示"+N 更多" |
| 年级 | 出现的年级 | 如：初三、初二 |
| 考试类型 | 考试类型 | 如：一模、二模、期中、期末 |
| 学期 | 学期 | 如：上学期、下学期 |

**表格交互：**
- 点击任意行 → 右侧滑出例句面板

**统计信息：**
- 显示当前筛选结果数量
- 筛选条件下显示"筛选结果"，无筛选显示"词库总量"

## 七、常见问题

### Q1：导入失败怎么办？

**可能原因及解决方案：**

| 原因 | 解决方案 |
|------|----------|
| 文件格式不正确 | 确保是.docx格式 |
| 文件损坏 | 重新保存或修复文件 |
| 文件名包含特殊字符 | 重命名文件，使用中文或英文字母 |
| 网络中断 | 检查网络后重试 |

### Q2：提取的词汇不准确怎么办？

词汇提取由AI完成，可能存在误差。如发现问题：
1. 核对原试卷内容是否清晰
2. 检查文件格式是否正确
3. 联系技术支持反馈问题

### Q3：如何更新已有试卷的词汇？
