## 测试步骤

### 1. 访问阅读页面

- [ ] 打开 http://localhost:5157/reading
- [ ] 页面正常加载
- [ ] 看到视图切换器（列表视图 / 讲义视图）位于页面右上角

### 2. 列表视图测试

- [ ] 默认显示列表视图（"列表视图" 按钮被选中）
- [ ] 筛选器正常工作（年级、区县、年份、考试类型、话题）
- [ ] 搜索框正常显示
- [ ] 文章列表正常显示（表格有数据）
- [ ] 点击文章可查看详情（右侧抽屉打开）
- [ ] 现有功能完全正常（无任何错误）

### 3. 视图切换测试

- [ ] 点击"讲义视图"按钮
- [ ] 按钮状态切换成功（"讲义视图" 被选中）
- [ ] 筛选器和表格消失
- [ ] 显示讲义视图占位符（显示"讲义视图"文字和图标）
- [ ] 切换回"列表视图"
- [ ] 列表视图恢复显示
- [ ] 筛选条件、分页状态保持正确

### 4. 边界情况测试

- [ ] 快速连续切换视图 5 次不会报错
- [ ] 视图切换时不会丢失数据
- [ ] 浏览器刷新后恢复默认视图（列表视图）
- [ ] 切换视图后，原有筛选条件仍然保留

### 5. C/D 篇切换测试（回归）

- [ ] 切换到"讲义视图"
- [ ] 再切换回"列表视图"
- [ ] 点击 C篇 Tab
- [ ] 表格只显示 C 篇文章
- [ ] 点击 D篇 Tab
- [ ] 表格只显示 D 篇文章
- [ ] 功能完全正常
